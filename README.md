# Simulador de Votación 🗳️

Este es un simulador de procesos electorales desarrollado con **FastAPI**.

## 🚀 Requisitos previos

- Python 3.x
- Git

## 🛠️ Instalación y Configuración

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com
   cd simulador_votacion
   ```

2. **Crear y activar el entorno virtual:**
   ```bash
   python -m venv .venv
   # En Windows (CMD):
   .venv\Scripts\activate
   ```

3. **Instalar dependencias:**
   ```bash
   pip install -r requirements.txt
   ```

## 💻 Ejecución

Para iniciar el servidor, ejecuta:
```bash
uvicorn main:app --reload
```
Luego ve a [http://127.0.0](http://127.0.0) para probar la API de forma interactiva.
